# Чат с шифрованием — руководство пользователя

Консольный мессенджер на FASM (x86-64, Linux) с RSA шифрованием.

> Работает только на Linux (или WSL на Windows).

---

## Полезные команды

```bash
# Сборка
make

# Удалить бинарники
make clean

# Перезапустить сервер (убивает старый процесс, освобождает порт, запускает новый)
make restart        # или: bash restart.sh

# Удалить все ключи (.key) и базу данных (data.bin)
make cleandata
```

---

## Шаг 1 — Запуск сервера

```bash
./build/server
```

Сервер запускается на порту **8080**. Держи его в отдельном терминале.
При перезапуске данные восстанавливаются из `data.bin`.

---

## Шаг 2 — Запуск клиента

```bash
./build/client
```

Клиент подключается к `127.0.0.1:8080`.

---

## Шаг 3 — Вход или регистрация

```
1. Войти  2. Регистрация:
```

- **Первый раз** — выбери `2`. Вводишь имя пользователя, клиент генерирует RSA-ключи, регистрирует тебя на сервере и сохраняет `username.key` локально.
- **Повторный вход** — выбери `1`. Вводишь имя — клиент загружает `username.key` и проходит авторизацию.

> Пустое имя пользователя — клиент завершается.

---

## Главное меню

```
=== МЕНЮ ===
1. Создать чат
2. Войти в чат
3. Пригласить в чат
4. Выйти
```

**1 — Создать чат** — вводишь название (Enter без текста = отмена). После создания показывается ID нового чата, затем возврат в меню.

**2 — Войти в чат** — показывает только твои чаты (в которые тебя пригласили). Вводишь ID или Enter для возврата в меню.

**3 — Пригласить в чат** — показывает твои чаты, вводишь ID чата и имя пользователя. Пользователь должен быть зарегистрирован, а ты должен быть членом чата.

**4 — Выйти** — закрывает соединение.

---

## Внутри чата

После входа в чат автоматически загружаются все сообщения в формате `имя: текст`. Снизу поле ввода:

```
Введите сообщение (Enter = выйти):
```

- **Текст** — отправить сообщение (до 252 символов), чат обновится
- **Enter (пусто)** — выйти в главное меню

---

## Полный пример

**Терминал 1 — сервер:**
```bash
./build/server
```

**Терминал 2 — Alice:**
```
./build/client
1. Войти  2. Регистрация: 2
Имя пользователя: alice
Зарегистрирован

=== МЕНЮ ===
Выбор: 1          ← создать чат
Название чата: mychat
→ id : 1 : название : mychat

Выбор: 3          ← пригласить Bob
ID чата: 1
Имя пользователя: bob

Выбор: 2          ← войти в чат
ID чата: 1
→ hello
→ Enter           ← выйти
```

**Терминал 3 — Bob:**
```
./build/client
1. Войти  2. Регистрация: 2
Имя пользователя: bob
Зарегистрирован

Выбор: 2          ← войти в чат
ID чата: 1
→ alice: hello
→ Enter           ← выйти
```

## Ограничения

| Параметр | Значение |
|---|---|
| Максимум пользователей | 16 |
| Максимум чатов | 16 |
| Максимум сообщений | 256 |
| Длина сообщения | до 252 символов |
| Длина имени пользователя | до 31 символа |
| Длина названия чата | до 31 символа |
